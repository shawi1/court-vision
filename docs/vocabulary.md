# Coach Vocabulary Glossary

> **Status:** Research pass v1 (2026-05-14). This document is load-bearing: the play schema (`docs/play-schema.md`) and the LLM system prompt are both derived from it. If the parser misses a term a real coach commonly uses, that play fails. Update this file whenever a coach uses a term we didn't catch.

Each entry follows:

```
### Term

**Aliases:** synonyms / alternative spellings
**Category:** screen | cut | location | inbound | formation | role | action | defense | tempo | set
**Definition:** coach-facing, one or two sentences
**Example:** "..." — what a coach might actually say
```

Categories are grouped under `##` headings; entries are alphabetized within each category.

## Screens

### Back Screen

**Aliases:** rip, rip screen, back pick
**Category:** screen
**Definition:** An off-ball screen set behind a defender so the cutter can attack the rim from the front. The screener faces the ball, the cutter cuts away from the screener toward the basket.
**Example:** "3 sets a back screen for 5, and 5 cuts to the rim for the lob."

### Ball Screen

**Aliases:** PnR, PNR, P&R, pick-and-roll, screen-and-roll, on-ball screen, pick
**Category:** screen
**Definition:** A screen set on the ball-handler's defender, the foundation action of modern basketball. The screener then rolls, pops, slips, or ghosts depending on the read.
**Example:** "5 comes up and sets a ball screen for 1 at the top, 1 rejects it and goes left."

### Brush Screen

**Aliases:** brush, brush cut, rub
**Category:** screen
**Definition:** A screen where the screener barely contacts the defender — they get in the defender's path without setting a true pick. Functions like a moving screen the ref won't call.
**Example:** "4 brushes 2's man on the way to the corner."

### Butt Screen

**Aliases:** flat screen, butt
**Category:** screen
**Definition:** A flat ball screen where the screener's back is square to the ball-handler (rather than angled to one side). Forces the defense to choose a side.
**Example:** "5 comes up and sets a butt screen at the nail."

### Cross Screen

**Aliases:** cross, X screen
**Category:** screen
**Definition:** An off-ball screen set horizontally, allowing a cutter to move from one side of the lane to the other — typically a big screening for another big across the paint.
**Example:** "4 sets a cross screen for 5, 5 cuts across to the ball-side block."

### DHO

**Aliases:** dribble handoff, handoff, hand-off, dribble hand-off
**Category:** screen
**Definition:** A dribble handoff — the ball-handler dribbles toward a teammate and hands the ball off; the handoff itself acts as a screen on the receiver's defender.
**Example:** "5 dribbles up, DHO to 2 on the wing, 2 turns the corner."

### Double Drag

**Aliases:** drag-drag, 77, 55, double-drag
**Category:** screen
**Definition:** Two staggered ball screens set by trailing bigs in transition or early offense. Stretches the defense across two screens before they've gotten matched up.
**Example:** "Push it, 4 and 5 set a double drag for 1 in the middle of the floor."

### Down Screen

**Aliases:** down, pin down, pindown, pin-down
**Category:** screen
**Definition:** An off-ball screen where the screener faces the baseline so the cutter comes from the block area up toward the perimeter — used to free shooters. Pin down and down screen are interchangeable in modern usage.
**Example:** "4 sets a down screen for 2 on the left side, 2 pops to the wing for the catch-and-shoot."

### Drag Screen

**Aliases:** drag, trail screen
**Category:** screen
**Definition:** An on-ball screen set by a trailing big in transition before the defense has gotten matched up. Almost always set near the top of the arc.
**Example:** "5 trails, drag screen for 1, attack the middle."

### Elevator Screen

**Aliases:** elevator, gate, double, doors
**Category:** screen
**Definition:** Two screeners stand apart with a gap between them; the cutter runs through the gap and the screeners then step together to close the doors on the trailing defender.
**Example:** "Elevator for 3 — 4 and 5 close the gate, 3 comes out for the three."

### Flare Screen

**Aliases:** flare, fade screen
**Category:** screen
**Definition:** An off-ball screen set so the cutter moves away from the ball, typically out to a perimeter spot for a catch-and-shoot. Different from a fade cut in that the screen, not the cutter alone, creates the separation.
**Example:** "Ball reversal to 4, 5 sets a flare for 2, throw it back to 2 for the open three."

### Ghost Screen

**Aliases:** ghost, slip ghost, blur, fake screen
**Category:** screen
**Definition:** A fake ball screen — the screener approaches as if to set a pick but cuts away (usually to the perimeter) before making contact. Punishes switches and aggressive coverages.
**Example:** "5 comes up and ghosts the screen, pops to the top for three."

### Hammer Screen

**Aliases:** hammer
**Category:** screen
**Definition:** A weak-side back/flare screen set in the corner while the ball-handler drives baseline strong side. The shooter receives the kick-out in the weak-side corner.
**Example:** "1 drives baseline, 4 sets a hammer for 3 in the weak-side corner, kick it for three."

### Pin-in Screen

**Aliases:** pin-in, pin in, low pin
**Category:** screen
**Definition:** An off-ball screen near the low block where the screener faces the basket (pinning the defender against the baseline) so the cutter pops toward the corner.
**Example:** "4 pins in 3's defender, 3 lifts to the corner."

### Ram Screen

**Aliases:** ram, ram action
**Category:** screen
**Definition:** A two-screen sequence where one player sets a screen for the eventual screener of the ball-handler — the "ram" frees the screener's defender so the upcoming ball screen catches them recovering.
**Example:** "3 rams 5's man, then 5 sets the ball screen for 1."

### Re-Screen

**Aliases:** rescreen, second screen, re-pick
**Category:** screen
**Definition:** When a screener resets and sets the screen a second time, typically because the defense recovered on the first attempt.
**Example:** "If they switch, 5 re-screens for 1."

### Ricky

**Aliases:** Ricky action, back-and-down, screen-and-rescreen
**Category:** screen
**Definition:** A back screen followed by a down screen for the same cutter, with the second screen set in the opposite direction of the first. Two screens, same cutter, same screener.
**Example:** "Ricky for 2 — 5 back screens for 2 to the rim, then immediately turns and down screens 2 back up to the wing."

### Slip

**Aliases:** slip the screen, slipping, early roll
**Category:** screen
**Definition:** When the would-be screener cuts to the basket instead of setting the screen, usually because the defense is trying to switch or trap. The screener never makes contact.
**Example:** "5 slips the screen and dives to the rim before they can switch."

### Spain Action

**Aliases:** Spain pick-and-roll, Spain PnR, stack PnR
**Category:** screen
**Definition:** A ball screen with a simultaneous back screen set by a shooter on the roller's defender; the back screener then pops out for three. Originated with Sergio Scariolo and Spain national team.
**Example:** "5 sets a ball screen for 1, 3 back-picks 5's man and pops to the top — Spain."

### Stagger

**Aliases:** stagger screen, staggered screens, double stagger
**Category:** screen
**Definition:** Two screens set in succession (not side by side) for the same cutter, in the same direction. The cutter weaves through both.
**Example:** "Stagger for 2 — 4 and 5 set staggered down screens on the left side."

### Step-Up Screen

**Aliases:** step-up, step up, side screen
**Category:** screen
**Definition:** A side ball screen set with the screener's back to the baseline, angling the ball-handler downhill toward the rim along the sideline.
**Example:** "5 steps up to screen for 1 on the wing, 1 attacks baseline."

### Twist

**Aliases:** twist screen, screen-the-screener
**Category:** screen
**Definition:** A screen-the-screener action — Player A screens for Player B, then Player C immediately screens for Player A. Often used to free the original screener as the actual shooter.
**Example:** "4 down screens for 2, then 5 twists and screens for 4 popping to the top."

### UCLA Screen

**Aliases:** UCLA, UCLA back screen, high post screen
**Category:** screen
**Definition:** A back screen set at the high post / elbow for a wing or guard who has just passed the ball, allowing them to cut to the rim. The signature action of John Wooden's UCLA offense.
**Example:** "1 passes to the wing, UCLA cut off 5 at the elbow."

### Wedge Screen

**Aliases:** wedge, angled back screen
**Category:** screen
**Definition:** An angled back screen set by a guard for a big cutting from the top to the weak-side low block, used to set up post-ups or wedge roll actions.
**Example:** "2 wedges for 5, 5 cuts to the weak-side block and seals."

## Cuts

### 45 Cut

**Aliases:** 45, 45-cut, diagonal cut
**Category:** cut
**Definition:** A diagonal cut at a 45-degree angle from the wing toward the basket, common in 5-out and read-and-react systems.
**Example:** "If 1 dribbles at 2, 2 makes a 45 cut to the rim."

### Backdoor Cut

**Aliases:** backdoor, back cut, back door
**Category:** cut
**Definition:** A cut behind the defender toward the rim, typically when the defender is over-playing the passing lane. The pass is a bounce or lob over the top.
**Example:** "If they top-lock 2, 2 backdoors and 1 hits him on the move."

### Banana Cut

**Aliases:** banana
**Category:** cut
**Definition:** A curved cut shaped like a banana, used to get a better passing angle when coming around a screen or cutting baseline.
**Example:** "3 banana cuts from the corner up to the wing."

### Baseline Cut

**Aliases:** baseline drift, drift, baseline runner
**Category:** cut
**Definition:** A cut along the baseline, often from the strong-side corner to the weak-side corner (or vice versa) — frequently the result of a baseline drive by the ball-handler.
**Example:** "1 drives middle, 3 baseline cuts from the strong-side corner."

### Basket Cut

**Aliases:** rim cut, cut to the rim
**Category:** cut
**Definition:** A direct, hard cut to the rim — no screen, no special angle. Often a give-and-go or the first cut of a Princeton-style offense.
**Example:** "2 passes and basket cuts, 5 hits him on the move."

### Curl

**Aliases:** curl cut, hard curl
**Category:** cut
**Definition:** A cut where the cutter curls tightly around the top of a screen and continues toward the rim, used when the defender trails over the screen.
**Example:** "If 2's man chases, 2 curls the down screen to the rim."

### Fade

**Aliases:** fade cut
**Category:** cut
**Definition:** A cut where the cutter moves away from the screen toward open space (typically the corner or wing) — used when the defender goes under the screen.
**Example:** "If they go under, 2 fades to the corner off the down screen."

### Flex Cut

**Aliases:** flex
**Category:** cut
**Definition:** A baseline cut where a perimeter player uses a cross screen from a low-post player to cut to the opposite block. Core action of the Flex offense.
**Example:** "3 flex cuts off 4's screen on the block."

### Give-and-Go

**Aliases:** pass and cut, give-go
**Category:** cut
**Definition:** Pass the ball, then immediately cut to the basket for a return pass. The oldest action in basketball.
**Example:** "1 hits 2 on the wing and give-and-gos to the rim."

### I Cut

**Aliases:** I-cut, I cut
**Category:** cut
**Definition:** A cut straight up the lane from the strong-side block to the strong-side elbow, used to clear baseline driving room for the ball-handler.
**Example:** "5 makes an I-cut to the elbow so 1 has the baseline."

### Iverson Cut

**Aliases:** Iverson, over the top, AI cut
**Category:** cut
**Definition:** A horizontal cut from one wing to the other across the top, usually over two staggered screens set at the elbows. Named for Allen Iverson.
**Example:** "Iverson for 2 — 4 and 5 at the elbows, 2 cuts across the top to the right wing."

### L-Cut

**Aliases:** L cut
**Category:** cut
**Definition:** A cut in the shape of an L — usually from the low block straight up the lane line to the elbow and then out to the wing.
**Example:** "3 makes an L-cut from the block out to the right wing."

### Laker Cut

**Aliases:** Laker, post-and-cut
**Category:** cut
**Definition:** A basket cut by the player who just passed the ball into the post — designed to occupy the post defender's help eyes and create a give-and-go look.
**Example:** "1 enters to 5 in the post and Laker cuts to the rim."

### Reject

**Aliases:** reject the screen, refuse
**Category:** cut
**Definition:** When the ball-handler or cutter chooses not to use a designed screen and instead attacks the opposite way — usually because the defense is overplaying.
**Example:** "5 comes to ball screen, 1 rejects and goes left."

### Replace Cut

**Aliases:** replace, fill, lift
**Category:** cut
**Definition:** When a player moves into a spot just vacated by a cutting teammate, maintaining offensive spacing.
**Example:** "When 2 basket cuts, 3 replaces him on the wing."

### Shuffle Cut

**Aliases:** shuffle, slice
**Category:** cut
**Definition:** A cut through the lane off a back screen set by a wing or post, usually from the top/slot area to the opposite block. Core action of the Shuffle offense.
**Example:** "4 sets a back screen at the elbow, 1 shuffle cuts to the weak-side block."

### UCLA Cut

**Aliases:** UCLA
**Category:** cut
**Definition:** The cut that comes off a UCLA screen — guard passes to the wing, then cuts off a high-post back screen to the rim.
**Example:** "1 enters to 3, UCLA cuts off 5 at the elbow."

### V-Cut

**Aliases:** V cut, jab cut
**Category:** cut
**Definition:** A perimeter cut shaped like a V — the player steps in toward the lane, plants, and explodes back out to receive a pass. The bread-and-butter cut to get open on the wing.
**Example:** "2 V-cuts and comes back to the right wing for the catch."

### Zipper Cut

**Aliases:** zipper, zip
**Category:** cut
**Definition:** A cut from the low block straight up the lane to the slot/top of the key, usually off a down screen on the lane line.
**Example:** "5 down screens, 1 zips up to the top for the catch."

## Court Locations

### Above the Break

**Aliases:** ATB, top three, above-the-break three
**Category:** location
**Definition:** Anywhere on the three-point arc above the corner — i.e., the slot, top of the key, and wing areas.
**Example:** "We want all our threes above the break or from the corner — nothing in between."

### Ball Side

**Aliases:** strong side, strongside
**Category:** location
**Definition:** The half of the court where the ball is currently located (split by the lane line extended).
**Example:** "On the ball side we want the corner filled."

### Block

**Aliases:** low block, post block
**Category:** location
**Definition:** The small marked rectangles on the side of the lane, about 4 feet from the baseline. Reference point for post-ups and low-post screens.
**Example:** "5 starts on the left block."

### Corner

**Aliases:** deep corner, three-point corner
**Category:** location
**Definition:** The spot where the baseline meets the sideline; in modern basketball, almost always referring to the corner three.
**Example:** "3 is in the right corner, 2 is in the left corner."

### Dunker Spot

**Aliases:** dunker, short corner area, below the corner
**Category:** location
**Definition:** A spot just outside the paint along the baseline, between the block and the short corner. Where the rolling big lives in 5-out offenses.
**Example:** "5 lives in the dunker spot when 1 drives."

### Elbow

**Aliases:** high post elbow, FT elbow
**Category:** location
**Definition:** Where the free-throw line meets the lane line — left elbow and right elbow. A key landmark for high-post action, UCLA screens, and Horns sets.
**Example:** "4 flashes to the left elbow."

### FT Line

**Aliases:** free-throw line, free throw line, foul line, charity stripe
**Category:** location
**Definition:** The free-throw line; the back of the paint. Often the spot for short rolls and high-post flashes.
**Example:** "5 short rolls to the free-throw line."

### Hash / Hash Mark

**Aliases:** hash, hash mark, sideline hash
**Category:** location
**Definition:** The hash marks on the sideline (28 feet from baseline at the NBA/college level). Used as the spot for inbounders on SLOB plays and as a reference for trigger actions.
**Example:** "1 is going to inbound from the hash."

### Help Side

**Aliases:** weak side, weakside
**Category:** location
**Definition:** The half of the court opposite the ball — where help defenders rotate from.
**Example:** "3 lifts up from the help side to the slot."

### High Post

**Aliases:** high post area, top of the key area
**Category:** location
**Definition:** The area around the free-throw line and the elbows — the "high" part of the post compared to the block.
**Example:** "5 flashes to the high post."

### Horns Spot

**Aliases:** horns elbows, horns
**Category:** location
**Definition:** The two elbows occupied by bigs in a Horns alignment. Used as shorthand: "both bigs at the horns."
**Example:** "4 and 5 are at the horns."

### Key

**Aliases:** the key, paint, lane area
**Category:** location
**Definition:** The painted rectangular area under the basket plus (informally) the free-throw circle on top. "In the key" usually means in the painted area.
**Example:** "Nobody in the key for the first three seconds."

### Lane Line

**Aliases:** key line, paint line
**Category:** location
**Definition:** The lines that bound the painted area on either side. A reference for zipper cuts and down screens (typically set on the lane line).
**Example:** "Set the screen right on the lane line."

### Low Post

**Aliases:** post, block area
**Category:** location
**Definition:** The area around the low block — the traditional post-up location.
**Example:** "5 ducks in to the low post and seals."

### Nail

**Aliases:** middle of the FT line, center of FT line
**Category:** location
**Definition:** The middle of the free-throw line — a defensive landmark (where the weak-side help defender often sits) and an offensive flash spot.
**Example:** "4 flashes to the nail."

### Near / Far

**Aliases:** ball-near, ball-far
**Category:** location
**Definition:** Side-relative shorthand. "Near" means same side as the ball; "far" means opposite side. Often used interchangeably with strong/weak side.
**Example:** "Near corner is filled by 3, far corner stays empty."

### Paint

**Aliases:** key, painted area, lane
**Category:** location
**Definition:** The painted rectangle under the basket. Same area as "the key" in casual usage.
**Example:** "5 ducks into the paint."

### Short Corner

**Aliases:** short, baseline gap
**Category:** location
**Definition:** The area along the baseline between the corner and the block — strategic spot against zones and a common skip/kick target.
**Example:** "4 flashes to the short corner against the zone."

### Slot

**Aliases:** slot area, top slot
**Category:** location
**Definition:** The two perimeter spots on the arc just to either side of the top of the key (lane lines extended out to the arc).
**Example:** "1 brings it up to the right slot, 2 fills the left slot."

### Top of the Key

**Aliases:** top, point, TOK
**Category:** location
**Definition:** The arc above the free-throw circle, dead center of the floor. The point guard's typical starting spot.
**Example:** "5 sets the ball screen at the top of the key."

### Weak Side

**Aliases:** weakside, help side, helpside
**Category:** location
**Definition:** The half of the court without the ball.
**Example:** "Hammer screen for 2 on the weak side."

### Wing

**Aliases:** wing area
**Category:** location
**Definition:** The area on the perimeter at the free-throw-line extended — between the slot and the corner.
**Example:** "2 spots up on the left wing."

## Inbound Terminology

### 5-Second Count

**Aliases:** 5 seconds, five count, inbound count
**Category:** inbound
**Definition:** The 5-second limit to inbound the ball after the official hands it to the inbounder. Failure is a turnover. Sets the urgency on every BLOB/SLOB play.
**Example:** "If nothing's open by three, just hit the safety to beat the count."

### BLOB

**Aliases:** baseline out of bounds, baseline inbound, baseline OB
**Category:** inbound
**Definition:** Baseline out-of-bounds play — the ball is inbounded from under or behind the offense's own basket.
**Example:** "Box BLOB — 4 and 5 on the strong-side block, 2 and 3 on the weak-side block."

### Inbounder

**Aliases:** inbound man, passer, trigger man
**Category:** inbound
**Definition:** The player throwing the ball in from out of bounds. On BLOBs they're under the rim; on SLOBs they're at or near the hash.
**Example:** "3 is the inbounder, 1 is the safety."

### Safety

**Aliases:** safety valve, outlet, release
**Category:** inbound
**Definition:** The designated bailout receiver on an inbound play — usually the point guard popping out to receive the ball if the primary read isn't there.
**Example:** "1 stays high as the safety on this BLOB."

### SLOB

**Aliases:** sideline out of bounds, sideline inbound, side OB, SOB
**Category:** inbound
**Definition:** Sideline out-of-bounds play — the ball is inbounded from the sideline, typically near the hash mark in the front court.
**Example:** "SLOB from the right hash — 5 down screens for 2."

### Trigger

**Aliases:** trigger action, key, initiator, first action
**Category:** inbound
**Definition:** The first action of an inbound play that starts everything else — usually a screen or cut by an off-ball player. The inbound pass typically goes to whoever the trigger frees up, or the trigger sets up the next action.
**Example:** "Trigger is 4 screening 3 to the corner."

## Formations

### 1-4 High

**Aliases:** 1-4-high, four high, 1-4 stack high
**Category:** formation
**Definition:** Point guard at the top, two wings at the free-throw-line extended, two bigs at the elbows. Opens the floor underneath for cuts and drives.
**Example:** "Start in 1-4 high, 1 dribbles at 2 for a DHO."

### 1-4 Low

**Aliases:** 1-4-low, four low
**Category:** formation
**Definition:** Point guard at the top, two players in the corners, two bigs on the low blocks. Common starting alignment for BLOB sets.
**Example:** "1-4 low — 5 cross screens for 4."

### Box

**Aliases:** box set, square
**Category:** formation
**Definition:** Four players forming a box — two at the elbows, two at the blocks. Common starting alignment for BLOBs and many half-court sets.
**Example:** "BLOB out of a box — 5 and 4 on the strong-side, 2 and 3 on the weak-side."

### Diamond

**Aliases:** diamond set, kite
**Category:** formation
**Definition:** One player at the top of the key, one at each elbow, and one under the basket — forming a diamond shape. Common BLOB alignment.
**Example:** "Diamond BLOB — 3 starts under the rim."

### Floppy

**Aliases:** floppy set, floppy formation
**Category:** formation
**Definition:** A shooter starts under the rim with a single screen on one side and a stagger on the other — the shooter chooses which side. Designed for elite shooters.
**Example:** "Floppy for 2 — single down on the right, stagger on the left, 2 picks his shot."

### Horns

**Aliases:** A set, A formation, horns set
**Category:** formation
**Definition:** Point guard at the top of the key, both bigs at the elbows, two shooters in the corners. The starting formation for hundreds of NBA sets.
**Example:** "Horns — 4 and 5 at the elbows, 1 with the ball at the top."

### Line

**Aliases:** stack line, BLOB line
**Category:** formation
**Definition:** A formation where players line up single-file along the lane line or the foul line — a common BLOB setup.
**Example:** "BLOB line — all four lined up on the right lane line."

### Pistol

**Aliases:** pistol set, 21
**Category:** formation
**Definition:** An early-offense alignment where the point guard dribbles up one side, a guard fills the strong-side corner, and a big trails — initiating a DHO or ball screen on the wing.
**Example:** "Pistol right — 1 dribbles right, 2 in the corner, 5 trails."

### Stack

**Aliases:** stack set, BLOB stack
**Category:** formation
**Definition:** Two or more players stacked one behind the other — most commonly a four-player stack on the lane line for BLOBs.
**Example:** "Stack BLOB — 4-5-2-3 stacked on the right lane."

### T

**Aliases:** T set, T formation
**Category:** formation
**Definition:** Three players across the free-throw line with one player at the top forming the stem of the T (or similar inverted variant). Used for BLOBs.
**Example:** "T set BLOB — three on the foul line, 1 at the top."

### Zipper

**Aliases:** zipper set
**Category:** formation
**Definition:** A set where the strong-side wing starts at the block and zipper-cuts up to the slot off a down screen on the lane line to start the offense.
**Example:** "Zipper — 2 starts on the block, 5 down screens, 2 zips up to the slot."

## Player References

### 1 (Point Guard)

**Aliases:** PG, point, point guard, ball-handler, one
**Category:** role
**Definition:** Positional role 1 — the primary ball-handler. In ATO design, "1" is the role, not the jersey number. Coaches design plays for roles; whichever player is at the 1 spot in that lineup runs that role.
**Example:** "1 brings it up and initiates."

### 2 (Shooting Guard)

**Aliases:** SG, two-guard, off-guard, two
**Category:** role
**Definition:** Positional role 2 — the off-guard, typically the team's primary perimeter scorer/shooter.
**Example:** "2 starts in the right corner."

### 3 (Small Forward)

**Aliases:** SF, wing, three
**Category:** role
**Definition:** Positional role 3 — the wing, a versatile perimeter player who can shoot, drive, and defend multiple positions.
**Example:** "3 lifts from the corner to the wing."

### 4 (Power Forward)

**Aliases:** PF, four, stretch four, combo big
**Category:** role
**Definition:** Positional role 4 — the power forward, traditionally a frontcourt player. In modern basketball, often a stretch shooter or screener at the elbow.
**Example:** "4 sets the screen at the elbow."

### 5 (Center)

**Aliases:** C, five, big, the big, the post
**Category:** role
**Definition:** Positional role 5 — the center, the rim-protector / primary screener / roller. Sets the ball screen on most pick-and-rolls.
**Example:** "5 comes up to set the ball screen for 1."

### Big

**Aliases:** bigs, frontcourt, post
**Category:** role
**Definition:** Generic term for a 4 or 5 — coaches often say "the big sets a screen" without specifying which one.
**Example:** "Both bigs at the horns."

### Guard

**Aliases:** guards, perimeter
**Category:** role
**Definition:** Generic term for a 1, 2, or 3 — a perimeter player.
**Example:** "The two guards exchange in the corners."

## Actions & Movements

### Ball Reversal

**Aliases:** reverse, swing, swing the ball
**Category:** action
**Definition:** Passing the ball from one side of the court to the other, usually through the top of the key. Forces the defense to shift and rotate.
**Example:** "Reverse it through 4 at the top to 2 on the weak-side wing."

### Cut Through

**Aliases:** through cut, clear cut
**Category:** action
**Definition:** A player cutting from the strong side through the paint to the weak side — usually to clear space and relocate.
**Example:** "1 cuts through to the weak-side corner to clear out the side."

### Duck-In

**Aliases:** duck in, seal
**Category:** action
**Definition:** A post move where the player seals their defender on the back hip using the inside foot — establishing position deep in the paint.
**Example:** "5 ducks in on the weak-side block."

### Exchange

**Aliases:** swap, switch (off-ball)
**Category:** action
**Definition:** Two off-ball teammates switch spots — used to confuse defensive matchups or relocate to new shooting positions.
**Example:** "3 and 2 exchange on the weak side."

### Flash

**Aliases:** flash cut, post flash, flash to the middle
**Category:** action
**Definition:** A quick cut by a post player from the weak side into the high post / nail / elbow area to receive a pass.
**Example:** "4 flashes to the nail for the touch."

### Kick-Out

**Aliases:** kick, kick out, drive-and-kick
**Category:** action
**Definition:** A pass from a driving player back out to a shooter on the perimeter.
**Example:** "1 drives baseline and kicks out to 3 in the corner."

### Pop

**Aliases:** pick-and-pop, pop out, screen and pop
**Category:** action
**Definition:** After setting a ball screen, the screener steps out to the perimeter (usually behind the arc) instead of rolling. The pop-out option in any PnR.
**Example:** "5 sets the screen and pops to the top."

### Re-pin

**Aliases:** repin, re-pin screen
**Category:** action
**Definition:** A second screen set immediately after the first — typically a re-screen on the same defender to free the shooter again.
**Example:** "5 down screens, defender chases, 5 re-pins on the recovery."

### Rim Run

**Aliases:** rim runner, sprint the rim, drag
**Category:** action
**Definition:** A big sprinting straight to the rim in transition — typically the trailing 5 who beats his defender down the floor.
**Example:** "5 rim-runs the right side, 1 hits him for the lob."

### Roll

**Aliases:** roll to the rim, dive
**Category:** action
**Definition:** After setting a ball screen, the screener cuts hard to the rim looking for a pocket pass or lob.
**Example:** "5 screens and rolls hard to the rim."

### Short Roll

**Aliases:** short, pocket
**Category:** action
**Definition:** Instead of rolling all the way to the rim, the screener stops short around the free-throw line, becoming a playmaker in the middle of the floor.
**Example:** "5 short rolls to the foul line, makes the play out of the pocket."

### Skip Pass

**Aliases:** skip, cross-court, skip-pass
**Category:** action
**Definition:** A long pass across the court that skips the next teammate — used to attack help defense and find weak-side shooters.
**Example:** "1 drives middle, skip pass to 3 in the weak-side corner."

### Slip (action)

**Aliases:** slip cut, early slip
**Category:** action
**Definition:** Same as the screen variant — the would-be screener cuts to the rim before contact. Listed here because coaches frequently say "slip" as a verb without referencing the screen.
**Example:** "5 slips it, dives to the rim."

### Swing

**Aliases:** swing pass, ball swing
**Category:** action
**Definition:** Pass the ball laterally to the next teammate on the perimeter. Often used during ball reversal.
**Example:** "Swing it to 4, then reverse to the weak side."

### Throw-back

**Aliases:** throwback, throw back, redirect
**Category:** action
**Definition:** A pass back to the player who just gave it up — typical of flare screens (drive, then throw back to the player who flared off the screen).
**Example:** "1 drives, throws back to 2 coming off the flare."

## Named Sets / ATO Concepts

### Chicago Action

**Aliases:** Chicago, Chi, pin-and-handoff
**Category:** set
**Definition:** A pin-down screen immediately followed by a dribble handoff — the receiver curls off the pin-down and gets handed the ball on the move. Synonymous with Zoom action in most coaches' usage.
**Example:** "Chicago for 2 — 4 pins down for 2, 5 hands off, 2 attacks downhill."

### Floppy

**Aliases:** floppy action
**Category:** set
**Definition:** A formation and action — the shooter starts under the rim and chooses to cut off the single screen on one side or the staggered screen on the other.
**Example:** "Floppy for 2, 2 reads which side has the better matchup."

### Get Action

**Aliases:** get, throw-and-get, chase handoff
**Category:** set
**Definition:** The point guard passes the ball ahead to a wing or big, then immediately chases for a handoff back — get the ball, give the ball, get the ball back.
**Example:** "1 throws ahead to 5 at the elbow, gets it back on the handoff — Get action."

### Hammer

**Aliases:** hammer set, hammer action
**Category:** set
**Definition:** A weak-side back screen set in the corner while the ball-handler drives baseline strong-side. Spurs-era staple action ending in a corner three.
**Example:** "Hammer — 1 drives baseline, 4 back screens for 3 in the weak corner."

### Horns Flare

**Aliases:** horns fade, A flare
**Category:** set
**Definition:** Out of Horns: one big sets a ball screen for the point guard, the other elbow big sets a flare screen for the weak-side shooter — drive option or kick-back to the flare.
**Example:** "Horns Flare — 5 ball-screens for 1, 4 flares for 2 on the right wing."

### Horns Out

**Aliases:** A out, stack out
**Category:** set
**Definition:** Out of Horns: one elbow big sets a cross screen for the other, who then flashes or rolls. Common ATO opener.
**Example:** "Horns Out — 4 cross screens for 5, 5 rolls to the strong-side block."

### Iverson Series

**Aliases:** Iverson, AI series
**Category:** set
**Definition:** Any set built off the Iverson cut — wing cuts across the top of two elbow screens before initiating the next action (DHO, ball screen, post-up).
**Example:** "Iverson into ball screen — 2 cuts across, 5 follows with the ball screen."

### Pistol

**Aliases:** pistol action, 21 series
**Category:** set
**Definition:** Early-offense action between three players — the point guard, a strong-side guard, and a trailing big. Reads include a DHO, a ball screen, or a quick post entry.
**Example:** "Pistol right — 1 dribbles at 2, 2 chooses DHO or backdoor."

### Punch Action

**Aliases:** punch, post-up call
**Category:** set
**Definition:** A direct entry to a post-up — typically a guard dribbles toward the post player, who flashes or seals; the entry pass goes inside.
**Example:** "Punch 5 — give 5 the ball on the left block, clear out."

### Quick Action

**Aliases:** quick, quick hitter
**Category:** set
**Definition:** A short, one-pass-and-shoot set designed for late-clock or sideline-out-of-bounds situations. Usually one screen, one cut, one shot.
**Example:** "Quick — DHO into a ghost screen, shoot it."

### Spain PnR

**Aliases:** Spain, Spain action
**Category:** set
**Definition:** A middle ball screen with a simultaneous back screen on the roller's defender, the back-screener popping for three. Stresses drop coverage.
**Example:** "Spain — 5 screens for 1, 3 back-picks 5's man and pops to the top."

### Stagger Step-Up

**Aliases:** stagger PnR, stagger into screen
**Category:** set
**Definition:** A staggered screen for a shooter that flows directly into a ball screen / step-up for the same player.
**Example:** "Stagger step-up — 4 and 5 stagger for 2 on the wing, then 5 step-ups for 2."

### Twist Action

**Aliases:** twist
**Category:** set
**Definition:** A screen-the-screener set where the original off-ball screener becomes the cutter freed by a second screen.
**Example:** "Twist — 4 down screens for 2, 5 twists and screens for 4 popping out."

### Wedge Action

**Aliases:** wedge roll
**Category:** set
**Definition:** A side pick-and-roll preceded by a wedge back screen from a guard on the big — the big gets a head start downhill before screening for the ball.
**Example:** "Wedge — 2 wedges for 5, 5 then ball-screens for 1 on the wing."

### Zoom

**Aliases:** zoom action
**Category:** set
**Definition:** A three-person action: down screen for the shooter immediately followed by a dribble handoff. The shooter ends up attacking downhill off the DHO. Often interchangeable with Chicago.
**Example:** "Zoom for 2 — 4 down screens, 5 hands off, 2 turns the corner."

## Defensive Coverages

### Blitz

**Aliases:** trap, hard trap, double
**Category:** defense
**Definition:** Both pick-and-roll defenders aggressively double-team the ball-handler off the screen, forcing a pass out.
**Example:** "If they blitz 1, 5 short-rolls into the pocket."

### Deny

**Aliases:** denial, top of the line
**Category:** defense
**Definition:** Defender positions in the passing lane to prevent the offensive player from catching the ball.
**Example:** "If they deny 2, run him back door."

### Drop

**Aliases:** drop coverage, sag, conservative
**Category:** defense
**Definition:** The screener's defender drops back below the level of the screen, protecting the rim and conceding mid-range pull-ups.
**Example:** "Against drop, 1 takes the pocket pull-up."

### Gap

**Aliases:** gap defense, sag, help posture
**Category:** defense
**Definition:** Off-ball defenders sag one or two steps off their man into the gap, ready to help on drives.
**Example:** "They're playing gap on 4 — pull-up for three."

### Hedge

**Aliases:** show, hard show, hedge-and-recover
**Category:** defense
**Definition:** The screener's defender briefly jumps out at the ball-handler to slow them down, then recovers to the roller.
**Example:** "5 sets the screen, if they hedge, 5 slips it before the recovery."

### ICE

**Aliases:** ice, down, blue, push
**Category:** defense
**Definition:** Side ball-screen coverage where the on-ball defender forces the ball-handler away from the screen and toward the baseline.
**Example:** "If they ICE the side ball screen, 1 rejects to the middle."

### Next / Tag

**Aliases:** next, nail-help, tag the roller
**Category:** defense
**Definition:** The weak-side or nail defender "tags" or "nexts" — bumping the roller momentarily on the way down so the screen defender can recover.
**Example:** "Hammer for 3 because they have to tag 5's roll."

### Over

**Aliases:** over the top, chase
**Category:** defense
**Definition:** The on-ball defender fights over the top of the ball screen — used against shooters who can pull up.
**Example:** "They go over, so re-screen and force a switch."

### Peel Switch

**Aliases:** peel, fly, scram switch
**Category:** defense
**Definition:** When a help defender switches onto the ball-handler driving past their man, the original defender peels off to pick up the open man.
**Example:** "Watch for the peel switch on the drive."

### Switch

**Aliases:** switch coverage, swap
**Category:** defense
**Definition:** The two defenders involved in a screen exchange men — the screener's defender takes the ball-handler and vice versa.
**Example:** "If they switch, slip the screen and attack the mismatch."

### Top-Lock

**Aliases:** top lock, high side
**Category:** defense
**Definition:** Off-ball defender positions on the high side of a shooter coming off a down screen, forcing them backdoor or away from their preferred catch.
**Example:** "They top-lock 2, so 2 backdoors and 1 hits him on the move."

### Under

**Aliases:** under coverage, go under
**Category:** defense
**Definition:** The on-ball defender goes underneath the ball screen — used against non-shooters.
**Example:** "If they go under, 1 re-screens or pulls up."

### Weak

**Aliases:** weak coverage, force weak
**Category:** defense
**Definition:** Coverage that forces the ball-handler to their weak hand — typically by positioning the on-ball defender on the strong-hand side.
**Example:** "They're forcing 1 weak, so set the screen on his left."

### X-Out

**Aliases:** X out, X rotation
**Category:** defense
**Definition:** A weak-side rotation where two defenders criss-cross their assignments after one helps on a baseline drive.
**Example:** "Baseline drive forces the X-out — kick to the open corner shooter."

## Tempo / Timing

### Early Offense

**Aliases:** early, early action, primary into early
**Category:** tempo
**Definition:** The first 6-10 seconds of the shot clock after pushing the ball up — before settling into the half-court set. Includes drag screens, pistol, and rim runs.
**Example:** "Push it, early offense — drag screen for 1."

### Late

**Aliases:** late clock, late-clock action
**Category:** tempo
**Definition:** A play designed for the last few seconds of the shot clock — usually a single quick action with a shooter as the read.
**Example:** "Late clock — ghost screen, 1 attacks for the pull-up."

### Quick

**Aliases:** quick hitter, quick action
**Category:** tempo
**Definition:** A short, one-action play designed to get a shot up in 5-7 seconds. Typical of ATOs.
**Example:** "Run Quick — ball screen, shoot it."

### Secondary Break

**Aliases:** secondary, second break
**Category:** tempo
**Definition:** The phase of transition after the primary fast break has been stopped — the offense flows directly into structured early-offense actions before the defense is set.
**Example:** "Secondary break into Horns."

### Transition

**Aliases:** fast break, primary break, push
**Category:** tempo
**Definition:** Possessions where the offense pushes the ball up the floor before the defense is set. Distinct from half-court offense.
**Example:** "Get a stop, push it in transition."

### Walk-Up

**Aliases:** walk it up, slow tempo
**Category:** tempo
**Definition:** Slow tempo — the point guard walks the ball up and the team gets into a half-court set rather than pushing in transition.
**Example:** "Late game, lead is safe, walk it up."

## Notable ATO patterns

These are example named ATOs a coach might call out. Each will become a few-shot example in the LLM system prompt: transcript -> parsed play schema.

### Horns Flare

A Horns alignment turned into a ball screen plus a weak-side flare — the basic two-way Horns set.
> "Horns Flare. 1 has it up top. 4 and 5 at the elbows. 5 comes up and sets a ball screen for 1 going right. 4 sets a flare for 2 on the left wing. 1 reads it — drive if 4's man helps, throw back to 2 if not."

### Floppy

A wide-open shooter selection set out of an under-the-rim start.
> "Floppy. 2 starts under the rim. 4 sets a single down screen on the right block. 5 and 3 set a stagger on the left. 2 picks his side. We want the shot."

### Stagger Step-Up

Two consecutive screens into a side ball screen for the same player — a flow set.
> "Stagger step-up. 4 and 5 stagger for 2 coming up the right side. 2 catches on the wing. 5 turns and step-ups for 2 going baseline. We want the corner three from 3 if they tag the roll."

### Spain PnR

The middle ball screen with the back screen on the roller's defender.
> "Run Spain. 1 brings it middle. 5 sets a ball screen up top. As 5 rolls, 3 back-picks 5's man and pops to the top. Three options: 1 attacks, 5 lobs, or 3 catches for three."

### Hammer

The Spurs staple — baseline drive plus weak-side flare for the corner three.
> "Hammer. 1 has it on the right wing, attacks baseline. 4 sets a hammer screen for 3 in the left corner. Kick out, 3 shoots the corner three."

### Quick Iverson

A late-clock Iverson cut into a quick ball screen.
> "Quick Iverson. 2 starts on the right wing. 4 and 5 at the elbows. 2 Iverson cuts across the top. 5 comes up and ball-screens for 2 on the left side. Attack downhill."

### BLOB Box Hammer

A baseline out-of-bounds play from a box set ending in a hammer screen for a corner three.
> "Box BLOB. 3 inbounds. 4 and 5 strong-side block and elbow, 2 weak-side block. On 'go,' 5 down screens for 2 going to the corner. 2 catches, drives the baseline. 4 hammers in the weak corner for 3. Corner three for 3."

### SLOB Zoom

A sideline out-of-bounds play using Zoom action.
> "SLOB Zoom. 1 inbounds at the right hash. 5 at the left elbow, 4 at the right elbow, 2 on the left wing, 3 in the right corner. 4 down screens for 2 coming up to the slot, 5 follows with the handoff. 2 turns the corner downhill. 1 inbounds and replaces."

### Horns Twist

A screen-the-screener action out of Horns, freeing the original screener for a shot.
> "Horns Twist. 4 and 5 at the elbows. 4 down screens for 2 in the corner. As soon as 4 screens, 5 twists and screens for 4 popping to the top. Skip pass from 1 to 4 for the three."

### Get Slip

A throw-and-get action where the screener slips on the follow-up.
> "Get. 1 throws ahead to 5 at the elbow. 1 chases for the handoff. As soon as 1 has it, 4 comes to ball screen — and 4 slips it. Lob for 4 at the rim."

## Open questions for the schema

These are issues that came up during this vocabulary pass and that `docs/play-schema.md` will have to grapple with as it firms up.

1. **Reads / options / counters.** Most named ATOs above contain at least two reads ("if they switch, slip; if not, curl"). The current v0.1 schema renders only the primary read. We will need either (a) a `reads` array on each action with a coverage trigger, (b) a `counter` field that produces an alternate play branch, or (c) flatten everything to the most likely outcome at parse time. Decision should come before the system prompt is written, because the LLM needs to be told whether to emit branches or pick one.

2. **Simultaneity granularity.** Hammer screens, Spain PnR, Horns Flare, and many BLOBs require two actions happening at the same time (drive + flare, ball screen + back screen). `tick: integer` represents this as same-tick events, but the natural-language input is often "5 ball screens for 1 and at the same time 3 back-picks 5's man." Consider adding optional `duration` ticks or a `group` id so renderable simultaneous actions don't accidentally collapse into "back-to-back at the same tick" events.

3. **Trigger actions for inbound plays.** BLOB/SLOB plays have a "go" call and a designated trigger (first cutter, first screen, inbound pass). The play schema currently has `situation: "BLOB" | "SLOB"` but no trigger concept. Suggest `meta.trigger: { type: "cut" | "screen" | "pass"; actor: PlayerId }` so the renderer knows where the play really begins.

4. **Defender modeling for coverages.** Many ATOs are designed AGAINST specific coverages (Spain attacks drop, Hammer attacks low-man rotation, Zoom attacks chase). The v1 schema treats defenders as optional. For the scouting layer to be useful, we likely need at least an `expectedCoverage` annotation per action — even if defenders aren't rendered, the parser should capture what the coach expects the defense to do.

5. **Player movement vs. screen vs. cut.** The schema currently has `move`, `screen`, `cut`, `pass`, `dribble`, `handoff`, `shot`. But "ricky" is a screen-then-rescreen by the same player on the same target. "Twist" is a screen-the-screener. These are higher-level patterns of multiple primitive actions. Decide whether the LLM should emit them as one `t: "namedAction"` or decompose into primitives. Recommendation: emit primitives, but also store the coach's verbal label (`meta.name`) for round-tripping.

6. **Ball position.** Currently implicit (follows the last passer/dribbler/handoff). This breaks down in two cases: (a) handoff chains where the ball changes hands rapidly, and (b) Spain/Hammer where the ball-handler keeps the ball while two other players do off-ball actions. Likely fine implicit for v1, but worth verifying with the few-shot test plays.

7. **Tempo as a modifier.** "Quick", "late", "early" are real coach words but they describe the play as a whole, not a single action. Recommend `meta.tempo: "quick" | "late" | "early" | "secondary" | "transition" | "walk-up"` rather than per-action.

8. **Named locations vs. coordinates.** The glossary lists ~25 named court locations. Recommend these become a closed enum (`NamedLocation` type) in the schema, with `{x, y}` only as a strict fallback. The Skia renderer can then map enum -> pixel coordinates with one lookup table — and the LLM has clear targets to emit.

9. **Defenders / coverages.** The defense vocabulary section is currently included for the scouting layer's future use. v1 schema does not need to encode defenders, but the LLM should still parse "if they switch" / "against drop" phrasing into a `notes` field or counters block, so the data isn't lost.

10. **Role abstraction vs. jersey numbers.** Coaches say "1" / "5" / "the four" interchangeably with player names ("LeBron / Steph / our four"). The schema uses `P1..P5` mapped to roles. The LLM system prompt must explicitly disambiguate: when the coach says "5," they almost always mean the role (the center), not jersey #5. Worth a few-shot example that calls this out.
